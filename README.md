# Zomato Restaurant Analytics Dashboard (Excel Project)

# Project Overview
This project is an interactive **Excel Dashboard** built using the Zomato restaurant dataset. The dashboard provides insights into restaurant performance, cuisine trends, customer engagement, pricing analysis and online delivery services across different cities and countries.

The project demonstrates strong skills in:
- Data Cleaning
- Data Analysis
- Pivot Tables
- Pivot Charts
- Dashboard Design
- Business Insights
- Interactive Reporting

---

# Dashboard Pages

## 1️ Executive Overview Dashboard
This dashboard provides high-level business insights.

### Key KPIs
- Total Restaurants
- Average Rating
- Total Votes
- Average Cost for Two

### Visuals Included
- Table Booking Analysis
- Online Delivery Analysis
- Top 10 Cuisines by Votes
- Restaurants by Price Range
- Top 10 Restaurants by Votes
- Monthly Votes Trend
- Geographic Map Analysis

---

## 2️ Detailed Insights Dashboard
This dashboard focuses on deeper restaurant analytics and customer behavior.

### Key KPIs
- Highest Rating
- Most Expensive Cuisine
- City with Highest Votes
- Online Delivery %

### Visuals Included
- Online Delivery by Average Rating
- Restaurants by Price Bucket
- Top Cities by Restaurant Count
- Top Cuisines by Average Rating
- Top Cuisines by Average Cost
- Top 10 Cities by Votes
- Cost vs Rating Scatter Plot
